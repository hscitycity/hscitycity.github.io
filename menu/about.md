# 화성형 팔란티어 개발일지

개발하면서 무엇을 시도했고, 무엇이 됐고 안 됐는지 남기는 기록. 결과물뿐 아니라 과정과 이유를 같이 남깁니다.

## 프로젝트

**화성형 팔란티어 시스템** — 행정안전부-NIA '2026년 공공부문 AI서비스 지원' 공모사업

경기도 화성시의 분절된 도시 데이터를 온톨로지 기반으로 통합하는 문제해결형 도시데이터 플랫폼 구축. 팔란티어(Palantir Technologies)의 아키텍처를 오픈소스 스택으로 한국 지방정부 환경에 맞게 구현한다.

## 기술 스택

**Beta v0.1 (현재)**: HTML + Vercel 서버리스 + Naver API + Claude AI

**목표 (3-Tier)**: React + Node.js + PostgreSQL + PostGIS

---

각 글은 보통 아래 구조를 따릅니다.

- **배경**: 무엇을 만들려고 했는지, 왜 필요했는지
- **시도**: 어떤 방법을 시도했는지 (실패한 시도 포함)
- **결과**: 최종적으로 무엇이 됐는지
- **교훈**: 다음에 비슷한 상황에서 기억할 점
